# Social Media Performance Dashboard

An interactive Business Intelligence dashboard that consolidates social media performance across Instagram, TikTok, and YouTube, built to track engagement, reach, and content performance over a 6-month period.

## Feature
- Overview Page – KPI summary, engagement trend, platform contribution, and top-performing content
- Platform Deep Dive Page – platform-specific KPIs, follower growth trend, views vs engagement scatter, and content type performance
- Interactive Filtering – slicers for Platform, Month, Content Type, and Date range across both pages
- Custom DAX Measures – Engagement Rate, Follower Growth %, Reach Rate, Share Rate, and 7-day rolling engagement average

## Skills
Power BI, DAX, Power Query, Data Modeling, Data Visualization
